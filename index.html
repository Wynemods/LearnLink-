Lesson 01: Introduction about XD

Get started with the basics of Adobe XD.

Overview

In this introductory lesson, we will cover the fundamental concepts of Adobe XD. You will learn about the user interface, how to create and manage artboards, and the basic tools for designing and prototyping. This lesson is designed for beginners and will provide a solid foundation for your journey into UI/UX design with Adobe XD. By the end of this lesson, you will be able to navigate the software with confidence and create a simple mobile app screen.
